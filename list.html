Working with Definition List

HTML
HTML is an Acronym that means Hypertext Markup Language. It is the Basic Language of the World Wide Web. It is Used in Creating the Structure of the Web. It is mad up tags. Tags are made up of angular braces with a special word that performs a specific task or creates a specific structure. To be a Web Developer you should understand HTML very well.
CSS
CSS means Cascading Style Sheet. It is used to Design web Applications to make them appealing and enhance the User Experience. Without CSS Web pages or Web Application woulf be very flat and Boring
